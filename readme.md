# AI Analytics POC

This project demonstrates a rule-based, explainable AI analytics system
that answers complex business questions over a structured JSON dataset.

## Features
- Natural language analytics over JSON data
- Deterministic filtering and aggregation
- OpenAI-powered explanations
- Token usage and cost tracking
- HTML report generation
- Model optimization (GPT-4.1 Mini / Nano)
